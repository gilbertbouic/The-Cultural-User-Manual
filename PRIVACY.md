# Privacy Policy

## Our Privacy Promise

- **Ad-Free Forever**: We will never show advertisements.
- **No Tracking**: We do not use analytics or tracking pixels.
- **Local Storage Only**: All your data, including quiz scores and progress, is stored on your own device. We do not have access to it.
- **No Account Required**: You can use this application without creating an account or signing in.
- **Open Source**: Our code is open for public review on [GitHub](https://github.com/gilbertbouic/Cultural-Assimilation-Manual). We believe in full transparency.

## What We Store Locally

To provide features like progress tracking, we use your browser's local storage to save the following:

- Quiz scores and completion status
- Selected role profile for personalized content
- Overall progress metrics and completed scenarios

This data never leaves your device and is not transmitted to any servers.

## Data Control

You have full control over this data. You can clear it at any time through:
- Your browser's settings (Clear browsing data > Cookies and site data)
- The "Reset Progress" button within the application (if available)
- Browser developer tools (Application/Storage tab)

## How to Use the Application

This is a static web application that runs entirely in your browser. Simply:
1. Open the application in any modern web browser
2. Select your region or country cluster
3. Explore cultural guidance, scenarios, and quizzes
4. Your progress is automatically saved locally

No configuration or setup is required to start using the application.

## Questions or Concerns

If you have questions about privacy or the application, please visit our [GitHub repository](https://github.com/gilbertbouic/Cultural-Assimilation-Manual) to open an issue or review the source code.
