## How to Use

1. Open `index.html` in a web browser
2. Select your destination country
3. Choose a category to learn about
4. Read the cultural guidelines and tips

## Running the App

Simply open the `index.html` file in any modern web browser. No server or installation required!

Alternatively, you can serve it locally:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .
```

Then visit `http://localhost:8000` in your browser.
